# 🐍 贪吃蛇大乱斗 — Snake Battle Royale

一款网页版贪吃蛇大乱斗游戏，你和 4 条 AI 蛇同场竞技，活到最后！

## 快速开始

直接用浏览器打开 \`贪吃蛇.html\` 即可开始游戏。

## 玩法

| 操作 | 按键 |
| --- | --- |
| 移动 | \`↑ ↓ ← →\` 或 \`W A S D\` |
| 暂停 | \`Space\` |
| 移动端 | 屏幕滑动 **或** 虚拟方向键 |

- 控制你的绿色蛇吃掉红色食物来增长身体、增加分数
- 和 4 条 AI 蛇（小红、小蓝、小黄、小紫）同场竞技
- 撞墙、咬到自己、或撞到其他蛇的身体都会被淘汰
- AI 蛇被淘汰时会掉落食物，场面越打越激烈
- 你的蛇阵亡后游戏结束，显示最终排名

## 功能特性

- 🤖 4 条 AI 蛇各有不同的性格颜色和名称
- 🧠 AI 采用贪心寻路算法，主动追逐食物、躲避障碍
- 💥 蛇与蛇之间碰撞检测（头撞头同归于尽、头撞身体则后者阵亡）
- 🍎 AI 蛇阵亡时身体会掉落食物，战场动态变化
- 🏆 实时排行榜显示存活状态和分数
- 🌟 深色星空背景，磨砂玻璃风格面板
- 🔊 Web Audio API 实时音效反馈
- 📱 响应式设计，支持手机触屏操作

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript（Canvas）
- 零外部依赖，单文件开箱即用
