# 🐍 贪吃蛇 — Snake Game

一款精美的单机版网页贪吃蛇游戏，开箱即玩，无需安装任何依赖。

## 快速开始

直接用浏览器打开 \贪吃蛇.html\ 即可开始游戏：

`ash
# Windows
start 贪吃蛇.html

# macOS
open 贪吃蛇.html
`

## 玩法

| 操作 | 按键 |
| --- | --- |
| 移动 | \↑ ↓ ← →\ 或 \W A S D\ |
| 暂停 | \Space\ |
| 移动端 | 屏幕滑动 **或** 虚拟方向键 |

- 控制蛇头吃到红色食物来增加分数和身体长度
- 撞墙或咬到自己身体则游戏结束
- 每吃 3 个食物游戏会逐渐加速，挑战你的反应极限

## 功能特性

- 🌟 深色星空渐变背景，磨砂玻璃风格面板
- 🎨 蛇身从亮绿到蓝绿的渐变过渡，蛇头带动态眼睛
- 🔊 使用 Web Audio API 的实时音效反馈（吃食物 / 游戏结束）
- 📱 响应式设计，手机端自动显示虚拟方向键并支持滑动控制
- 💾 最高分自动保存在浏览器本地存储中
- ⏸ 空格键随时暂停

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript（Canvas）
- Web Audio API — 游戏音效
- localStorage — 最高分持久化
- 零外部依赖，单文件开箱即用
